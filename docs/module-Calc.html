    This method sums two(2) or more numbers together and returns the sum.

    Parameters:
    Name Type Description
    numbers Array.<Number> | Number

    The numbers to be summed up. Could be an array of numbers.
    Source:
    • tooljs.esm.js, line 4657
    Returns:

    The result of the summation

    Type
    Number
    Example 
    var numbers = [4, 4, 10];

Calc.add(numbers); // returns 18;
Calc.add(1, 2, 3, 4); // returns 10;
Calc.add(numbers, 1, 2, 3, 4, [5, 5], 2); // returns 40;

    Parameters:
    Name Type Description
